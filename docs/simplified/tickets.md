## How to Create a Ticket (Step by Step)

### Step 1: Click "Create Ticket"
Look for the blue "Create Ticket" button at the top of the tickets page.

### Step 2: Fill in the Form

**Title (Required)**
- Keep it short and clear
- Max 200 characters
- Good example: "Feed dispenser stuck in Tank 3"
- Bad example: "Problem" (too vague!)

**Description (Required)**
- Explain in detail
- Include all important information
- For issues: What happened? When did it start?
- For requests: What do you want? Why would it help?

**Good Description Example:**
```
The feed dispenser in Tank 3 has been stuck since this
morning at 9:00 AM. When I try to open it for feeding,
nothing happens. The tank ID is BF-TANK-003.

I checked:
- Power connection (looks fine)
- Control panel (shows green light)
- Other dispensers (they work fine)

This is blocking our feeding schedule.
```

**Type (Required)**
Choose one:
- âœ… Issue (something broken)
- âœ… Feature Request (want something new)
- âœ… Upgrade (make something better)
- âœ… Others (everything else)

### Step 3: Click "Submit"
Your ticket is created! You'll see it in "My Tickets" right away.

## What Happens After You Create a Ticket?

### Step 1: Ticket Created (You)
- Status: Open
- Priority: Unassigned
- You get confirmation

### Step 2: Admin Reviews (Admin)
- Admin sees your ticket
- Reads description
- May ask questions in comments
- Sets priority based on importance

### Step 3: Work Begins (Admin)
- Status changes to "In Progress"
- Admin investigates or develops
- May post updates in comments
- You can answer questions

### Step 4: Issue Resolved (Admin)
- Fix is implemented
- Status changes to "Resolved"
- Admin may ask you to test it

### Step 5: Confirmation (You)
- You test the fix
- Comment to confirm it works
- Or report if still broken

### Step 6: Ticket Closed (Admin)
- Status becomes "Closed"
- Admin adds closing comment
- Ticket is finalized
- All done! ğŸ‰

## Tips for Good Tickets

### âœ… DO:

**Be Specific**
- Bad: "Tank not working"
- Good: "Tank 3 water sensor not recording temperature"

**Include Details**
- Tank ID or number
- When problem started
- What you already tried
- Error messages you saw

**Be Polite**
- Remember: admins are helping you!
- Use please and thank you
- Be patient

**One Issue Per Ticket**
- Don't combine multiple problems
- Create separate tickets for different issues
- Easier to track and fix

**Check First**
- Look at existing tickets
- Someone may have reported it already
- Comment on existing ticket instead of duplicating

### âŒ DON'T:

**Don't Be Vague**
- "Something's wrong" - too vague!
- "It doesn't work" - what doesn't work?
- "Fix this" - fix what exactly?

**Don't Demand**
- Not: "FIX THIS NOW!!!"
- Better: "This is affecting our feeding schedule, can you help?"

**Don't Spam**
- One ticket is enough
- Don't create duplicates
- Use comments for updates

**Don't Include Passwords**
- NEVER put passwords in tickets
- Ask admin for secure way to share if needed

## Common Questions

### "How long until my ticket is fixed?"

It depends on:
- **Priority:** Critical issues are handled first
- **Complexity:** Some fixes are quick, others take time
- **Resources:** Admins work on multiple tickets

**What you can do:**
- Check the status for updates
- Read comments from admin
- Be patient
- If urgent, mention why in the description

---

### "Can I change the priority myself?"

**No.** Only administrators can set priority. They decide based on:
- How many people are affected
- Impact on operations
- Safety or revenue concerns
- Other urgent issues

---

### "My ticket is still 'Open' after 2 days, is that normal?"

Yes, it can be normal if:
- It's lower priority
- Admin is working on critical issues first
- More information is needed
- It's being reviewed

**If concerned:**
- Add a comment asking for an update
- Be polite
- Don't spam multiple comments

---

### "Can I delete my ticket?"

**No.** Only admins can close tickets. But you can:
- Comment saying you want it closed
- Explain why (problem solved, no longer needed, etc.)
- Admin will close it

---

### "I created a ticket by mistake, what do I do?"

**Option 1:** Add a comment saying it was a mistake and ask admin to close it.

**Option 2:** If you haven't submitted yet, just close the dialog and don't submit.

---

### "The issue came back after the ticket was closed"

**Create a NEW ticket!**
- Mention it happened before
- Include the old ticket number if you remember
- Explain what's different this time

Example:
```
Title: Feed dispenser stuck again in Tank 3

Description: This is the same issue as ticket #42 which was fixed
last week. It started happening again this morning. The dispenser
was working fine for 5 days after the fix.
```
